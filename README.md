# Can Preseason Climate Signals Predict Extreme California Wildfire Seasons?

## Project Overview

California has experienced several extreme wildfire seasons in recent years, creating major environmental, economic, and public safety challenges. This project explores whether preseason climate conditions can help predict how severe the upcoming wildfire season will be.

Rather than attempting to predict individual fires, this project focuses on forecasting **season-level wildfire threat** each year (for example: mild, moderate, severe, or extreme).

## Main Research Question

Can preseason climate indicators such as drought conditions, temperature anomalies, precipitation levels, snowpack, and ENSO signals help predict the severity of California wildfire seasons?

## Why This Matters

Earlier wildfire season forecasts could help improve:

- insurance risk planning  
- utility preparedness  
- vegetation management  
- emergency resource allocation  
- public awareness and planning  

## Planned Data Sources

- NOAA climate and temperature records
